# Coach E Coaching Website

A modern fitness coaching portfolio built with React and FastAPI. It showcases my coaching journey, client transformations, training approach, and the Coach E AI chatbot.

## Features

- Responsive fitness coaching portfolio
- About and fitness journey sections
- Anonymous client transformation showcase
- Infinite client results carousel
- Coach E AI chatbot
- Contact form and social links
- Scroll reveal and neon-style animations

## Tech Stack

- React
- Vite
- CSS
- FastAPI
- Groq API
- React Icons
